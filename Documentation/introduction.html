Introduction to Component Library

This Component Library is an open source styling framework, which helps Developers to use best styles and components, by using pre-defined classnames.
Component Library helps you to quickly build your Dream Project, by focusing much on Functionality, because we take care of your Designs.

Installation Guide

In order to introduce this Component Library to your project, Copy the Link given below and paste it in the <head> tag of your html, above all other style sheets.

Customizing Themes
