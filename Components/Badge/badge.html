Badge

Badges are small circles, positioned either at top-right or bottom-right of the parent component. Badges are being used to display a notification count or status information

Profile Badge

Status badges can be used to denote the status of a user. For example, it can show if the user is Online, Offline, Idle and dnd mode.


Number Badge

Number badges can be used to denote the numeric value of an entity i.e to show the no. of items present in the parent component.
