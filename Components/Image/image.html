Image

Image can be responsive to fit the parent's width, and also can be customised to be round shaped

Full Width Image

A Full Width Image will adapt to the width of it's parent container. Resize the brower window to see the image behave responsive. You can set custom width of the resp-img class, and the img will be responsive w.r.t to it.


Round Image

You can add class img-round to make your image with same width and height to round shaped.
