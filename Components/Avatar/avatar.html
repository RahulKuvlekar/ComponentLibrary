Avatar

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Circle Avatar

Circle Avatars can be created using the class 'avatar', inside the img tag.

Sizes Avatar

Size of the avatars can be changed by adding avatar-sm | avatar-md | avatar-lg | avatar-xlg class to the existing avatar class.

Square Avatar

Adding a class 'avatar-square' to existing class, will generate a square shaped avatar.
