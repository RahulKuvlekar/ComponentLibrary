Input

Inputs are necessary for user interaction with the website. Inputs are used to take information from the user, and then save the data to the server, accordingly.

Text Box

A textbox is used to take input from user. It can have different types of styles depending on the input given. For example: success, warning, error.


Textbox with validation (Error)

This style is used to depict that the data entered is incorrect.
*Wrong Password. Try again.


Textbox with validation (Success)

This style is used to depict that the data entered is correct.
*Strong password.


Textbox with validation (Warning)

This style is used to warn the user against a particular situation.
*Weak Password.
