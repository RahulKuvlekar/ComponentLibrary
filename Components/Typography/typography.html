Typography

Typography defines the font size, weight, etc. to be used throughout the project to maintain consistency.

Headings

For heading u can use h1, h2, h3, h4, h5, h6 elements.
The same font-styling is present for class names h1, h2, h3, h4, h5, h6 class. You can add one of these classes to style the text.

Paragraph Text

For paragraph u can use p element.
Other than this if you want to adjust text to small, medium or large, you can use text-lg, text-md, text-sm class.

Grey Text

Grey color fonts are easier on the eyes on a white background.
If you want to use different shades of Gray you can use text-grey-lt, text-grey-md, text-grey-lt class.
